Angular 18 Course

  • 1. Angular Architecture Overview (ng new <ProjectName>)
    Learn about Angular's architecture, modules, components, and how to create a new project using Angular CLI.
  • 2. Components Life Cycle Hooks(ng g c <ComponentName>)
    Understand Angular component lifecycle hooks like ngOnInit, ngOnChanges, and ngOnDestroy.
  • 3. Routing - Create Routes, Navigation
    Set up routing, create routes for components, and implement navigation between views.
  • 4. Data Binding - One Way, Two Way
    Learn one-way binding and two-way binding.
  • 5. Control Flow Statement - if, for
    Use Angular's control flow statements for conditional rendering and looping.
  • 6. Forms - Template / Reactive
    Build forms using template-driven and reactive approaches, and implement form validation.
  • 7. API Integration - httpClient, GET, POST, PUT, DELETE
    Integrate APIs using HttpClient and perform CRUD operations.
  • 8. Services(ng g ervices <ServiceName>)
    Create and use services to share data and functionality across components.
  • 9. Pipes
    Use built-in pipes and create custom pipes to transform data in templates.
  • 10. Component Input, Output
    Pass data between parent and child components using Input and Output.
  • 11. Signal
    Learn Angular's new signal feature for reactive state management.
  • 12. Advance Topics
    Explore advanced concepts like lazy loading, route guards, interceptors, and state management.
  • 13. Small Projects
    Build small projects like a to-do list, weather app, or e-commerce product listing to apply your knowledge.
